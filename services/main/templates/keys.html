How to use your Keys

Import your keys into Australia Cash Core wallet

  1. In the running AusCash client, click help/debug window/console.

  2. If your wallet is encrypted type walletpassphrase "your password" 600

  3. type importprivkey

  4. Example: importprivkey 5KJvsngHeMpm884wtkJNzQGaCErckhHJBGFsvd3VyK5qMZXj3hS
